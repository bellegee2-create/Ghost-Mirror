# Ghost-Mirror


A Firefox extension that mirrors a claude.ai conversation to a detachable, 
scrollable window. Built to solve the lack of a native persistent conversation 
view in claude.ai for long technical sessions.

## Features
- Full conversation mirror in a separate draggable window
- All/Human/Claude filter, search, auto-scroll
- Download proxy — click any file tag to trigger download silently
- Works for all file types (zip, docx, py, txt, js, bat, md etc.)
- Live sync as conversation streams

## Install
1. Download the zip from Releases
2. Firefox: about:addons → gear → Install Add-on From File

## Status
Firefox only. Chrome port pending.
